## 📝 Introduce

다이어트에 필수적인 식단관리. 

사진을 찍어 올리면 칼로리를 계산해줍니다!

하루동안 섭취한 음식의 칼로리를 확인하고 식단을 조절해보세요.

- 직접 칼로리를 입력 ❌
- 앱 내 검색으로 입력 ❌

## ⭐ Main Feature

- 이미지 분석으로 사진 속 음식들 칼로리 계산.

## 🔧 Stack

**Java Springboot**

- JPA
- MySQL
- Thymeleaf

**Python Flask**

- Yolo v5
- BeautifulSoup
